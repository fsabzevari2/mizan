Mizan License Manager - راه‌انداز سریع

مراحل نصب و راه‌اندازی:
1) پوشه mizan-license را در مسیر wp-content/plugins/ قرار دهید.
2) افزونه را از منوی افزونه‌ها در وردپرس فعال کنید.
3) مطمئن شوید SSL فعال است (HTTPS).
4) در اپ از مسیر REST API زیر برای ارسال درخواست ثبت‌نام استفاده کنید:
   - ثبت درخواست: POST https://cofeclick.ir/wp-json/mizan/v1/register
     body JSON: { "email","first_name","last_name","username","phone","store_name","device_hash" }
   - بررسی وضعیت: POST https://cofeclick.ir/wp-json/mizan/v1/check
     body JSON: { "device_hash": "..." } یا { "email": "..." }

جریان کار پیشنهادی:
- کاربر در اپ ثبت‌نام می‌کند -> درخواست به /register ارسال می‌شود (status = pending)
- در وردپرس به منوی "Mizan Licenses" بروید، درخواست را تأیید کنید (دکمه تأیید)
- افزونه یک کلید لایسنس تولید و به ایمیل کاربر ارسال می‌کند؛ اعتبار پیش‌فرض 14 روز است
- کاربر در اپ با خواندن پاسخ /check متوجه می‌شود لایسنس دارد یا خیر
- پس از دریافت مبلغ کارت‌به‌کارت، شما در پنل روی همان ردیف تمدید یا دائمی می‌کنید (Extend)

نکات امنیتی:
- فایل private.pem (اگر خواستید JWT یا امضا اضافه کنید) را همیشه خارج از public_html قرار دهید.
- برای نسخهٔ بعدی: میتوانم JWT با امضا RS256 اضافه کنم تا اپ به‌صورت آفلاین هم امضا را بررسی کند.
